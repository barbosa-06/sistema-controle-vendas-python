# 🛒 Sistema de Controle de Vendas em Python

Projeto desenvolvido em Python para gerenciamento de produtos, controle de estoque e registro de vendas.

## 🚀 Funcionalidades

* Cadastro de produtos
* Listagem de produtos cadastrados
* Controle de estoque
* Registro de vendas
* Atualização automática do estoque
* Relatório de faturamento
* Produto mais vendido
* Persistência de dados utilizando arquivo TXT

## 🛠️ Tecnologias utilizadas

* Python
* Programação Orientada a Objetos (POO)
* Manipulação de Arquivos TXT
* Listas e Dicionários
* Tratamento de Exceções

## 📋 Como executar

1. Clone o repositório:

```bash
git clone https://github.com/barbosa-06/sistema-controle-vendas-python.git
```

2. Acesse a pasta do projeto:

```bash
cd sistema-controle-vendas-python
```

3. Execute o programa:

```bash
python main.py
```
